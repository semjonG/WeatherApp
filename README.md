# WeatherApp

Simple Weather App on SwiftUI with openWeather API. 

## Pay attention

To build this code you should add your APIKey from https://openweathermap.org 

```swift
struct WeatherAPI {
    static let apiKey = "PUT HERE YOUR API KEY"
}
```

## Used

- MVVM
- SwiftUI
- CoreLocation
- Lottie Animation
